DateJS

is an incredibly fast, lightweight, and easy-to-use date library

Installation

DateJS is so popular that it comes pre-installed with all javascript environments (including your browser) and can be accessed through the Date constructor!

Usage

As mentioned previously, you would just want to use the Date constructor.
So, you would simply do: 

const date = new Date();
